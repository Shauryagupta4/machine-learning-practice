# House Price Prediction using Linear Regression

A beginner machine learning project built using **Scikit-Learn** to predict house prices based on different house features.

## Tech Stack

* Python
* Pandas
* NumPy
* Scikit-Learn

## Features

* Handles missing values
* Encodes categorical data using One-Hot Encoding
* Splits data into training and testing sets
* Standardizes numerical features
* Trains a Linear Regression model
* Predicts house prices
* Evaluates model performance using:

  * Mean Absolute Error (MAE)
  * Mean Squared Error (MSE)
  * Root Mean Squared Error (RMSE)
  * R² Score

## Dataset

This project uses a small sample dataset created for learning purposes.

Features include:

* Area
* Bedrooms
* Bathrooms
* Age
* Parking
* Location

Target:

* House Price

## Project Workflow

1. Load dataset
2. Handle missing values
3. Encode categorical features
4. Split data into training and testing sets
5. Scale numerical features
6. Train Linear Regression model
7. Make predictions
8. Evaluate model performance
